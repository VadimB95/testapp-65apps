### 65apps staff
Приложение для отображения информации о персонале (тестовое задание).

Используется:
* MVVM
* Retrofit (HTTP запросы)
* Moshi (парсинг JSON)
* Room (SQLite)
* Hilt (Dependency Injection)
* LocalDateTime backport (работа с датой)
* SpinKit (анимация загрузки изображения)
* Timber (логирование)
* Glide (загрузка изображений по сети)

На данный момент есть небольшая проблема с минимальной версией API. Временно поднял до 21, т.к. Retrofit имеет особенности, связанные с обратной совместимостью. Как исправлю, сделаю коммит.
